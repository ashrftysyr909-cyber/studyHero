# بطل الدراسة - Study Hero 🎓🏆

تطبيق بطل الدراسة التفاعلي المتميز والكامل المبني باستخدام بيئة **Flutter** وهندسة الكود النظيف مع دعم كامل وتلقائي لتخطيط الـ RTL (من اليمين لليومين).

## متطلبات التشغيل (Requirements)
* تثبيت **Flutter SDK** (إصدار 3.0.0 أو أحدث).
* تثبيت **Android SDK** لبناء تطبيقات الأندرويد.

## خطوات البدء السريع (Quick Start)

1. قم بفك ضغط ملف المشروع بالكامل.
2. افتح مجلد المشروع في برنامج **VS Code** أو **Android Studio**.
3. قم بتشغيل الأمر التالي في سطر الأوامر لتحميل حزم الاعتماديات:
   ```bash
   flutter pub get
   ```
4. قم بربط هاتفك الأندرويد أو تشغيل محاكي، ثم شغّل التطبيق عن طريق:
   ```bash
   flutter run
   ```

## بناء ملف الـ APK النهائي للإنتاج (Build APK)
لتوليد ملف الـ APK النهائي القابل للتثبيت المباشر على الهواتف:
```bash
flutter build apk --release
```
ستجد الملف الناتج في المسار:
```
build/app/outputs/flutter-apk/app-release.apk
```

## هيكل المجلدات (Folder Structure)
* **lib/core/theme/**: يحتوي على تصميم وألوان وثيمات التطبيق المتناسقة والخطوط العربية الأنيقة.
* **lib/data/models/**: نماذج البيانات والأسئلة والتقدم.
* **lib/data/repositories/**: مستودعات الأسئلة وتخزين البيانات.
* **lib/providers/**: مزودي الخدمة وإدارة الحالة التفاعلية (النقاط، المستويات، المشتريات، الإنجازات).
* **lib/views/**: شاشات واجهة المستخدم المتكاملة (الترحيب، تسجيل الدخول، الرئيسية، الاختبار، المتجر، المتصدرين، الإحصائيات، الإعدادات).
